## Sobre

Loja virtual moderna desenvolvida com Next.js, focada em catalogo de produtos, busca e experiencia de navegacao.

O objetivo deste repositorio e demonstrar uma implementacao clara, organizada e facil de avaliar, com documentacao objetiva para recrutadores e pessoas tecnicas entenderem rapidamente o contexto do projeto.

## Funcionalidades

- Catalogo de produtos.
- Busca.
- Pagina de produto.
- Carrinho.
- Validacao de ambiente.
- Interface responsiva.

## Stack

- **Next.js 15**
- **React 19**
- **TypeScript**
- **Zod**
- **Tailwind CSS**
- **Lucide React**

## Como executar

```bash
git clone https://github.com/marquesmaycon/devstore.git
cd devstore
npm install
npm run dev
```

## Scripts disponiveis

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`

## Destaques tecnicos

- Projeto de e-commerce enxuto.
- Boa vitrine de Next.js e UI.
- Estrutura clara para catalogo e compra.
